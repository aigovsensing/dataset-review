오픈 데이터셋 법적 리스크 검토 에이전트

데이터셋을 킁킁대며 라이선스 · 데이터 생성/수집 방식 · 개인정보 포함 여부의 법적 리스크를 자동 검토하여 GitHub 이슈에 결과를 등록합니다. 🔍

아래 정보를 입력하고 검토 요청을 누르면 GitHub 이슈 작성 페이지가 열립니다. 내용을 확인한 뒤 Submit new issue 버튼을 누르면 검토가 자동으로 시작됩니다. (GitHub 로그인이 필요합니다.)

이용 방법

        
    
          
  1. 검토 요청 탭에서 데이터셋 명칭과 관련 URL을 입력합니다.
    2. 
          
  2. 검토 요청 버튼을 누르면 GitHub 이슈 작성 페이지가 자동으로 채워진 채 열립니다.
    3. 
          
  3. 내용을 확인하고 Submit new issue를 누르면 이슈가 생성됩니다.
    4. 
          
  4. GitHub Actions가 이슈를 감지하여 Gemini(Google AI Studio) + Google 검색으로 법적 리스크를 검토합니다.
    5. 
          
  5. 검토 결과가 해당 이슈에 댓글로 자동 등록됩니다. 검토 결과 탭에서 목록을 확인할 수 있습니다.
    6. 
        

        

          ⚠️ 본 검토 결과는 회사 내부 사전 리스크 검토용 참고 자료이며,
          법률 자문이나 법적 판단을 대체하지 않습니다. 확인되지 않은 사항은 "확인 불가"로 표기되며,
          이 경우 내부 검토 기준에 따라 잠재적 법적 리스크가 있는 것으로 판단합니다.
